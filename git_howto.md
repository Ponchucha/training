# Шпаргалка по GIT
## Инициализация нового репозитория
```sh
git init
```
## Получить информацию от GIT о его текущем статусе

```sh
git status
```

## Добавить файл или файлы к следубщему коммиту

```sh
git add
```

## Создание коммита
```sh
git commit -m "message"
```

## Вывод на экран истории всех коммитов с их хеш-кодами

```sh
git log
```
