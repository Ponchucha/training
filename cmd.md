# Подсказка по командной строке

## Переход в нужную директорию

### Windows

```sh
cd C:\users....\lesson01
```

## Проверка содержимого директории

### Windows

```sh
dir
```

### Linux/MacOs

```sh
ls
```

## Очистка поля командной строки

### Windows

```sh
cls
```

### Linux/MacOs

```sh
clear
```

## Удаление файла

### Windows

```sh
del file.txt
```

### Linux/MacOs

```sh
rm file.txt
```
